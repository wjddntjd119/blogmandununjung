# Components 폴더
1. BlogEditor
 ㄴ 게시글 작성하는 코드
2. BlogItem
 ㄴ 게시글 리스트에 띄울 아이템 저장
3. BlogList
 ㄴ BlogItem에 저장된 게시글들을 보여줌
 ㄴ 수정하기, 삭제하기 기능
4. Header
 ㄴ 맨위에 고정되는 헤더부분
 ㄴ 누르면 홈으로 가는 로고와, 로그인한 사람의 이름을 띄움.
